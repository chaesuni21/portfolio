## 1. JSP 회원가입 페이지
- HTML, CSS, JSP를 활용한 간단한 회원가입 웹페이지  
- **기능:** 사용자가 이름, 아이디, 비밀번호를 입력하면 서버에서 처리하여 환영 메시지 출력  
- **기술스택:** Java (JSP), HTML, CSS  
- **개발환경:** Eclipse, Apache Tomcat  
- **상태:** 기본 동작 구현 중
  
## 2. Unity 2D 플랫포머 게임 (Pixel Adventure Game)
- Unity 2D를 활용하여 제작한 횡스크롤 플랫포머 게임  
- **설명:** 플레이어가 점프와 이동을 통해 장애물을 피하고, 하트·열쇠·코인을 수집하며 스테이지를 클리어하는 구조  
- **기능:** 캐릭터 이동 및 점프, 아이템 수집, 게임오버/클리어 UI, 타이머  
- **기술스택:** Unity, C#
